02.Projetos

General questions

General questions é uma página web que contém três caixas de perguntas. Conforme o usuário clica no botão de +, abre uma extensão da caixa com a resposta. Nesse projeto utilizei alguns métodos e propriedades como: querySelectorAll, forEach, addEventListener, currentTarget, parentElement, classList, toggle.

Modal project

Esse projeto contempla um modal. Modal é um elemento utilizado em muitos sites, na maioria das vezes para mostrar informações que não estão na página.

Menu digital

Esse projeto consiste em um menu digital com quatro categorias de comidas: breakfast, dessert, dinner e lunch. Cada categoria possui imagens e alguns detalhes. Implementei alguns métodos para manipular array como: map, reduce, push e filter.
